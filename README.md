## 前言

```--陈伟平 记录于：2020年1月11日
	从我毕业工作现在已经有半年的时间了。从开始的磕磕绊绊一路走到现在，也算是入了Java这个行业的大门。在这半年中遇到了很多问题，也接触到了很多新的知识领域。
	现在想来，很后悔没有在之前的时间里对遇到的问题进行记录，这就导致一段时间之后很多当时已解决的问题，变的模糊不清甚至已经完全忘记了。所以再经过了数次花费大量时间去解决同一个问题的不爽经历之后，我下定决心开始记录我在工作中遇到的问题以及我的解决方案。
	其一是为了在面临同样的问题的时候不在做重复的劳动。其二则是想在之后的时间里或者再次遇到同一问题时能否对这个方案进行优化或者更好的解决。
```
